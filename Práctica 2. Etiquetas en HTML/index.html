ETIQUETAS

    Encabezados

  1. <h1> a <h6>: Etiquetas de encabezado que definen diferentes niveles de títulos en una página web. <h1> es el título principal y <h6> es el menos importante.

    2. Ejemplo:
    <h1>Este es un encabezado de nivel 1</h1>
    <h2>Este es un encabezado de nivel 2</h2>
    <h3>Este es un encabezado de nivel 3</h3>
    <h4>Este es un encabezado de nivel 4</h4>
    <h5>Este es un encabezado de nivel 5</h5>
    <h6>Este es un encabezado de nivel 6</h6>

    Párrafos y texto

  2. <p>: Etiqueta de párrafo para agrupar texto juntos.

    3. Ejemplo:
    <p>Este es un ejemplo de un párrafo de texto en HTML. Puedes utilizar este elemento para representar cualquier tipo de texto, desde descripciones hasta contenido de artículos y más. El elemento "p" también es útil para separar diferentes secciones de texto en tu página, lo que ayuda a mejorar la legibilidad y organización de tu contenido.</p>

  3. <br>: Salto de línea forzado.

    4. Ejemplo:
    <br/>

  4. <hr>: Línea horizontal para separar secciones en la página.

    5. Ejemplo:
    <hr/>

  5. <a>: Hipervínculo o enlace

    6. Ejemplo:
    <a href="https://www.ejemplo.com">Este es un enlace de ejemplo></a>

  6. <em>: Texto en cursiva.

    7. Ejemplo:
    <p>Este es un <em><strong>ejemplo</strong></em> de un texto enfatizado y resaltado.</p>

  7. <strong>: Texto en negrita.

    8. Ejemplo:
    <p>Este es un <strong> <em>ejemplo </em> </strong> de un texto enfatizado y resaltado. </p>

  8. <mark>: Texto resaltado.

    9. Ejemplo:
    <p>Este es un <mark>ejemplo</mark> de un texto marcado.</p>

  9. <small>: Texto en tamaño pequeño.

    10. Ejemplo:
    <p>Este es un ejemplo de <small>texto de menor tamaño</small> en una página web.</p>

  10. <sub>: Texto subíndice.

    11. Ejemplo:
    <p>La fórmula química del agua es H2O.</p>

  11. <sup>: Texto superíndice.

    12. Ejemplo:
    <p>La ecuación de la relatividad general es E = mc2.</p>

    Imágenes y multimedia

  12. <img>: Insertar una imagen en la página.

    13. Ejemplo:
    <img src="ruta/a/la/imagen.jpg" alt="texto alternativo">

  13. <audio>: Reproducir audio en la página.

    14. Ejemplo:
    <audio src="ruta/al/archivo-de-audio.mp3" controls> </audio>

  14. <video>: Reproducir video en la página.

    15. Ejemplo:
    <video src="ruta/al/archivo-de-video.mp4" controls> </video>

  15. <source: Define la ubicación de los recursos de audio o video para <audio> y <video>.

    16. Ejemplo:
    <source src="ruta/al/archivo-de-video-o-audio.mp4" type="tipo/mime-del-archivo">

  16. <track>: Define pistas de texto para <audio> y <video>.

    17. Ejemplo:
    <track src="ruta/al/archivo-de-subtitulos.vtt" kind="tipo-de-pista" srclang="idioma-del-subtitulo" label="etiqueta-del-subtitulo">

    Listas

  17. <ul>: Lista no ordenada.

    18. Ejemplo:
    <ul>
    <li>Manzanas</li>
    <li>Naranjas</li>
    <li>Plátanos</li>
    <li>Mangos</li>
    </ul>

  18. <ol>: Lista ordenada.

    19. Ejemplo:
    <ol>
    <li>Primer elemento de la lista</li>
    <li>Segundo elemento de la lista</li>
    <li>Tercer elemento de la lista</li>
    </ol>

  19. <li>: Elemento de lista.

    20. Ejemplo:
    <li>Primer elemento de la lista</li>
    <li>Segundo elemento de la lista</li>
    <li>Tercer elemento de la lista</li>

    Tablas

  20. <table>: Crea una tabla.

    21. Ejemplo:
    <table>
    <tr>
    <th>Nombre</th>
    <th>Edad</th>
    </tr>
    <tr>
    <td>Juan</td>
    <td>30</td>
    </tr>
    <tr>
    <td>María</td>
    <td>25</td>
    </tr>
    </table>

  21. <caption>: Añade una leyenda a la tabla.

    22. Ejemplo:
    <caption>Lista de productos </caption>


  22. <thead>: Crea el encabezado de la tabla.

    23. Ejemplo:
    <table>
    <thead>
    </table>

  23. <tbody>: Crea el cuerpo de la tabla.

    24. Ejemplo:
    <table>
    <thead>
    </tbody>
    </table>


  24. <tfoot>: Crea el pie de la tabla.

    25. Ejemplo:
    </tbody>
    <tfoot>
    <tr>
    <td colspan="3">Fin de la tabla</td>
    </tr>
    </tfoot>

  25. <tr>: Fila de la tabla.

    26. Ejemplo:
    <tr></tr>

  26. <th>: Celda de encabezado de la tabla.

    27. Ejemplo:
    <th>Nombre</th>

  27. <td>: Celda de datos de la tabla.

    28. Ejemplo:
    <td>John</td>

    Formularios

  28. <form>: Crea un formulario para enviar datos al servidor.

    29. Ejemplo:
    <form action="procesar_formulario.php" method="post"> <label for="nombre">Nombre: </label>
    </label>

  29. <input>: Crea un campo de entrada para que los usuarios ingresen información.

    30. Ejemplo:
    <input type="text" id="nombre" name="nombre">


  30. <textarea>: Crea un campo de entrada de texto de varias líneas.

    31. Ejemplo:
    <textarea id="comentario" name="comentario" rows="4" cols="40"></textarea>

  31. <select>: Crea un menú desplegable para que los usuarios seleccionen una opción.

    32. Ejemplo:
    <label for="frutas">Frutas:</label>
    <select id="frutas" name="frutas">
    </select>

  32. <option>: Define una opción para un menú desplegable.

    33. Ejemplo:
    <option value="manzana">Manzana</option>
    <option value="banana">Banana</option>
    <option value="naranja">Naranja</option>
    <option value="uva">Uva</option>

  33. <button>: Crea un botón para enviar un formulario.

    34. Ejemplo:
    <button>Presionar</button>

  34. <label>: Etiqueta para un campo de entrada, proporciona una descripción del campo.

    35. Ejemplo:
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre">

  35. <fieldset>: Agrupa elementos de formulario relacionados.

    36. Ejemplo:
    <fieldset></fieldset>

  36. <legend>: Añade una leyenda a un <fieldset>.

    37. Ejemplo:
    <legend>Información personal:</legend>

    Elementos semánticos

  37. <article>: Es un artículo completo o una sección de un artículo.

    38. Ejemplo:
    <article>
    <h2>El título de mi artículo</h2>
    </article>

  38. <aside>: Define contenido relacionado pero separado del contenido principal.

    39. Ejemplo:
    <aside>
    <h3>Barra lateral</h3>
    <p>Este es un contenido adicional relacionado con la sección principal.</p>
    </aside>

  39. <header>: Define la cabecera de una página o sección.

    40. Ejemplo:
    <header>
    <h1>Mi sitio web</h1>
    <nav>
    <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Acerca de</a></li>
    </ul>
    </nav>
    </header>

  40. <footer>: Define el pie de página de una página o sección.

    41. Ejemplo:
    <footer>
    <p>Derechos de autor © 2023 - Mi sitio web. Todos los derechos reservados.</p>
    </footer>

  41. <nav>: Define una sección de navegación.

    42. Ejemplo:
    <nav></nav>

  42. <main>: Define el contenido principal de una página.

    43. Ejemplo:
    <main></main>

    Otras etiquetas útiles

  43. <div>: Se utiliza como un contenedor genérico para otros elementos.

    44. Ejemplo:
    <div>
    <h2>Sección 1</h2>
    <p>Este es el contenido de la sección 1.</p>
    </div>

  44. <span>: Se utiliza para agrupar elementos en línea.

    45. Ejemplo:
    <p>Este es un ejemplo de un texto que contiene un <span>texto destacado</span>.



  45. <style>: Define estilos CSS en la página.

    46. Ejemplo:
    <style>
    body {
    background-color: #f5f5f5;
    font-family: Arial, sans-serif;
    }
    </style>

    Metadatos

  46. <head>: Contiene información sobre la página, como el título y los metadatos.

    47. Ejemplo:
    <head>
    <title>¡Hola, mundo!</title>
    </head>

  47. <title>: Define el título de la página que aparece en la pestaña del navegador.

    48. Ejemplo:
    <title>Programacion Python</title>

  48. <meta>: Proporciona información adicional sobre la página, como el autor, la descripción y las palabras clave.

    49. Ejemplo:
    <meta charset="UTF-8">
    <meta name="description" content="Un ejemplo de página web">

    Elementos de Programación

  49. <script>: Define un script de JavaScript que se ejecutará en la página.

    50. Ejemplo:
    <script>
    document.getElementById("saludo").innerHTML = "¡Hola, mundo!";
    </script>

  50. <noscript>: Proporciona contenido alternativo para los usuarios que tienen JavaScript desactivado.

    51. Ejemplo:
    <noscript>
    <p>Lo siento, su navegador no admite JavaScript.</p>
    </noscript>

    Atributos

  51. id: Identifica un elemento único en la página.

    52. Ejemplo:
    <header id="encabezado">
    <h1>Este es el encabezado de la página</h1>
    </header>

  52. class: Identifica un conjunto de elementos que comparten las mismas características.

    53. Ejemplo:
    <div class="mi-clase">
    <p>Este es un párrafo con la clase "mi-clase".</p>
    </div>

  53. style: Define estilos CSS para un elemento individual.

    54. Ejemplo:
    <p style="color: blue; font-size: 18px;">Este es un párrafo con estilo en línea.</p>

  54. src: Define la URL de la fuente para una imagen, audio, video u otro recurso.

    55. Ejemplo:
    <img src="mi-imagen.jpg" alt="Descripción de la imagen">

  55. href: Define la URL de destino para un enlace.

    56. Ejemplo:
    <a href="https://www.ejemplo.com">Este es un enlace a Ejemplo.com</a>

  56. alt: Proporciona texto alternativo para una imagen en caso de que no se pueda mostrar.

    57. Ejemplo:
    <img src="mi-imagen.jpg" alt="Descripción de la imagen">

    Etiquetas en HTML5

  57. <section>: Define una sección de contenido.

    58. Ejemplo:
    <section>
    <h2>Mi sección</h2>
    <p>Este es el contenido de mi sección.</p>
    <p>Este es otro párrafo en la sección.</p>
    </section>

  58. <figure>: Define una figura que puede incluir una imagen, un diagrama u otro contenido.

    59. Ejemplo:
    <figure>
    <img src="mi-imagen.jpg" alt="Descripción de la imagen">
    </figure>

  59. <figcaption>: Define el título de una figura.

    60. Ejemplo:
    <figcaption>Esta es la leyenda de la imagen.</figcaption>


  60. <time>: Define una fecha y hora.

    61. Ejemplo:
    <p>La fecha actual es <time datetime="2023-02-21">21 de febrero de 2023</time>.</p>

  61. <datalist>: Crea una lista de opciones para un campo de entrada.

    62. Ejemplo:
    <datalist id="opciones">
    <option value="Opción 1">
    <option value="Opción 2">
    <option value="Opción 3">
    <option value="Opción 4">
    </datalist>

  62. <meter>: Crea una barra de progreso para mostrar el estado de un valor numérico.

    63. Ejemplo:
    <label for="mi-medida">Mi medida: </label>
    <meter id="mi-medida" min="0" max="100" value="50">50 out of 100 </meter>

  63. <progress>: Crea una barra de progreso para mostrar el progreso de una tarea.

    64. Ejemplo:
    <label for="mi-tarea">Mi tarea:</label> <progress id="mi-tarea" value="50" max="100">50%</progress>

    Etiquetas de enlaces

  64. <link>: Define la relación entre el documento actual y un archivo externo, como una hoja de estilo CSS.

    65. Ejemplo:
    <link rel="stylesheet" type="text/css" href="mi-estilo.css">

  65. <area>: Define una región de un mapa de imagen y un enlace a otro documento o recurso.

    66. Ejemplo:
    <area shape="rect" coords="0,0,50,50" href="pagina1.html" alt="Área 1">

    Etiquetas multimedia adicionales

  66. <picture>: Proporciona una forma de mostrar imágenes responsivas.

    67. Ejemplo:
    <picture> <source media="(min-width: 800px)" srcset="imagen-grande.png">
    <source media="(min-width: 400px)" srcset="imagen-mediana.png">
    <img src="imagen-pequena.png" alt="Mi imagen">
    </picture>

  67. <embed>: Inserta contenido multimedia en una página.

    68. Ejemplo:
    <embed src="mi-video.mp4" type="video/mp4" width="640" height="360">

  68. <object>: Define un objeto incrustado en una página.

    69. Ejemplo:
    <object data="mi-video.mp4" type="video/mp4" width="640" height="360">
    <a href="mi-video.mp4">Descargar video </a>
    </object>

    Etiquetas de encabezado adicionales

  69. <hgroup>: Agrupa varios encabezados <h1> a <h6> juntos para formar un encabezado de nivel superior.

    70. Ejemplo:
    <hgroup>
    <1>Título principal</h1>
    <h2>Subtítulo</h2>
    </hgroup>

    Etiquetas de texto adicionales

  70. <cite>: Define una referencia a un trabajo o fuente.

    71. Ejemplo:
    <p>Según <cite>Wikipedia</cite>, HTML es el lenguaje de marcado estándar para crear páginas web.</p>

  71. <code>: Define una sección de código de programación.

    72. Ejemplo:
    <p>Para imprimir un mensaje en JavaScript, se utiliza <code>console.log('Hola, mundo!');</code>.</p>

  72. <pre>: Muestra el texto con el formato y los espacios en blanco tal como están en el archivo fuente.

    73. Ejemplo:
    <p>El siguiente código muestra cómo imprimir un mensaje por consola en JavaScript:</p>
    <pre>console.log('Hola, mundo!');</pre>

  73. <q>: Define una cita corta que se integra con el texto circundante.

    74. Ejemplo:
    <p>El poeta dijo: La vida es un sueño.</q></p>

  74. <blockquote>: Define una sección de texto que se cita de otra fuente.

    75. Ejemplo:
    <blockquote>
    "La ciencia es el gran antídoto contra el veneno del entusiasmo y la superstición."
    <br>
    <cite>Adam Smith</cite>
    </blockquote>

    Etiquetas de lista adicionales

  75. <dl>: Define una lista de definiciones.

    76. Ejemplo:
    <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language, lenguaje de marcado para la web.</dd>
    </dl<

  76. <dt>: Define un término en una lista de definiciones.

    77. Ejemplo:
    <dl>
    <dt>Manzana</dt>
    <dd>Fruta dulce de piel roja o verde y pulpa blanca.</dd>
    </dl>

  77. <dd>: Define un definición en una lista de definiciones.

    78. Ejemplo:
    <dl>
    <dt>Perro</dt>
    <dd>Mamífero domesticable, compañero y mascota fiel del ser humano.</dd>
    </dl>

    Etiquetas de tabla adicionales

    79. Ejemplo:
    <col style="width: 20%;">
    <col style="width: 60%;">
    <col style="width: 20%;">

  79. <colgroup>: Agrupa varias columnas en una tabla.

    80. Ejemplo:
    <colgroup>
    <col style="width: 20%;">
    <col style="width: 60%;">
    <col style="width: 20%;">
    </colgroup>

  80. <datalist>: Define una lista de opciones para un campo de entrada.

    81. Ejemplo:

    Atributos adicionales

  98. contenteditable: Permite al usuario editar el contenido de un elemento.

    99. Ejemplo:
    <p contenteditable="true">Este párrafo es editable. Puedes hacer clic en él y escribir cualquier cosa que desees.</p>

  99. spellcheck: Especifica si el contenido de un elemento debe ser verificado ortográficamente.

    100. Ejemplo:
    <textarea id="mensaje" name="mensaje" spellcheck="true"></textarea>

  100. required: Especifica que un campo de entrada debe ser completado antes de enviar el formulario.

    101. Ejemplo:
    <input type="text" id="nombre" name="nombre" required>

  101. autocomplete: Especifica si un campo de entrada debe tener autocompletado habilitado.

    102. Ejemplo:
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" autocomplete="on">

  102. placeholder: Define un texto de marcador.

    103. Ejemplo:
    <label for="buscar">Buscar:</label>
    <input type="text" id="buscar" name="buscar" placeholder="Ingresa tu búsqueda aquí">
